# איקס עיגול אונליין - Tic Tac Toe Online

משחק איקס עיגול מרובה משתתפים בזמן אמת עם תמיכה ב-WebSockets.

## תכונות

✅ **משחק בזמן אמת** - שני משחקנים משחקים באותו הזמן עם עדכונים מיידיים
✅ **הצטרפות קלה** - יצירת משחק וקבלת קישור לשיתוף עם חברים
✅ **בחירת שמות** - כל שחקן בוחר את שמו לפני המשחק
✅ **ניהול תורות** - רק השחקן שתורו עומד בתורו יכול לעשות מהלך
✅ **ניקוד** - צבירת ניקוד לאורך המשחקים
✅ **איפוס** - אפשרות לאפס משחק חדש או לאפס את כל הניקוד

## דרישות

- Node.js (גרסה 14 ומעלה)
- npm

## התקנה והפעלה

### התקנת Node.js (אם לא מותקן)

הורד ותקן Node.js מהאתר הרשמי: https://nodejs.org/

### הפעלת המשחק

**Windows:**
לחץ פעמיים על הקובץ `start.bat`

**macOS/Linux:**
```bash
chmod +x start.sh
./start.sh
```

**או באופן ידני:**
```bash
cd TicTacToe
npm install
npm start
```

השרת יפעל על `http://localhost:3000`

### פתיחת המשחק

פתח את הדפדפן שלך וכנס לכתובת: `http://localhost:3000`

## איך משחקים

### יצירת משחק חדש
1. הכנס את שמך
2. לחץ על "צור משחק חדש"
3. שתף את הקישור או קוד המשחק עם חברך
4. כאשר חברך הצטרף, המשחק יתחיל

### הצטרפות למשחק קיים
1. הכנס את שמך
2. לחץ על "הצטרף למשחק"
3. הכנס את קוד המשחק (6 תווים)
4. כאשר הצטרפת, המשחק יתחיל

### כללי משחק
- השחקנים מוקצים כ-X ו-O באופן אקראי
- כל שחקן משחק בתורו בלבד
- השחקן הראשון מתחיל תמיד בתור הראשון של משחק חדש
- ניצחון: 3 בשורה (אופקי, אנכי, או אלכסוני)
- תיקו: מלוא הלוח ללא ניצחון
- הניקוד צובר עם כל משחק עד לאיפוס

## פתרון בעיות

### "ספריית התקשורת לא נטענה"
- וודא שהשרת פועל (`npm start`)
- רענן את הדף (F5)

### "אין חיבור לשרת"
- בדוק שהשרת פועל בטרמינל
- וודא שאין חומת אש שחוסמת את הפורט 3000

### "התקשורת עם השרת נותקה"
- בדוק את חיבור האינטרנט
- הפעל מחדש את השרת
